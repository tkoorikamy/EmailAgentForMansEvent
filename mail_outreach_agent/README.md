# Mail Outreach Agent

Windows 10 desktop-приложение для легальной B2B email-рассылки коммерческих предложений с обязательным предпросмотром.

## Установка
```bash
cd mail_outreach_agent
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

## Запуск
```bash
cd mail_outreach_agent
python -m app.main
```

## Сборка EXE
```bat
cd mail_outreach_agent
build_windows.bat
```
Готовый файл будет в `dist/`.

## Формат таблицы получателей
Обязательные колонки:
- `company`
- `email`

Необязательные:
- `contact_name`
- `category`
- `city`
- `comment`
- `website`

Поддерживаются `.xlsx` и `.csv`.

## Подключение SMTP
Во вкладке **Почтовый ящик**:
1. Заполните host/port/SSL/login/password.
2. Нажмите **Проверить подключение**.
3. Нажмите **Сохранить**.

Пароль хранится через `keyring` (Windows Credential Manager), не в открытом виде.

## Пошаговый сценарий работы
База → КП и шаблон → Сгенерировать письма → Предпросмотр → Выбрать письма → Отправить выбранные → Отправка и логи → Старт.

1. **База**: импортируйте таблицу, проверьте статусы (`ready`, `invalid_email`, `duplicate`, `skipped`).
2. **Почтовый ящик**: настройте SMTP и сохраните, при необходимости нажмите **Отправить тестовое письмо**.
3. **КП и шаблон**: загрузите файл КП (PDF/DOCX/XLSX), отредактируйте тему/тело, нажмите **Сгенерировать письма**.
4. **Предпросмотр**: проверьте письма, выберите получателей и нажмите **Отправить выбранные**.
5. **Отправка и логи**: нажмите **Старт** для запуска очереди, отслеживайте статус и экспортируйте отчет CSV.

## Безопасность и ограничения
- Нет скрытой массовой отправки без предпросмотра.
- Есть подтверждение перед отправкой выбранных получателей.
- Строки с ошибками email не должны отправляться.
- Лимиты и задержки на отправку поддерживаются.
- Footer с отказом добавлен в шаблон.

## AI-персонализация
- Вкладка **КП и шаблон** поддерживает 2 режима: обычный шаблон и AI-персонализация.
- Для AI установите переменную окружения `MAIL_OUTREACH_AI_API_KEY` (ключ не хранится в коде и файлах настроек).
- Для AI-писем предпросмотр обязателен: откройте карточку получателя и сохраните перед отправкой.
- Если ключ не задан или AI недоступен, приложение автоматически продолжит работу через обычный шаблон.
